Lost World SMP Guide

How to Link Your Minecraft and Discord Accounts

To link your Minecraft and Discord accounts, follow these steps:

  1. Join the server: Connect to the Minecraft server using the IP lwsmp.mooo.com.
  2. Register/Log in: Once you've logged in or registered, run the command /discord link.
  3. Receive the code: You’ll get a 4-digit code in-game.
  4. DM the bot: Send this code to the bot "@Lost World SMP" via direct message on discord.
  5. Accounts linked: Once the code is sent, your Minecraft and Discord accounts will be linked.

Note: Linking your accounts is not required to join the server, but it is highly recommended for a better experience.

Account Types

To determine the type of account you have:

  • Premium Account: If you've paid for your account and it isn’t a Bedrock account, you have a premium account.

    • Important: If you have a premium account, it's highly recommended to run the command /premium after joining the server. Failing to do so could result in losing all your items, and you’ll be forced to log in every time you join.

  • Cracked or Bedrock Account: If you’re using a cracked or Bedrock account, you can skip this step.

How to Play

To join the server:

  1. Start the server: In Discord, use the command !startserver to start the Minecraft server.
  2. Join the server: Once the server has started, connect with the IP lwsmp.mooo.com.

Important: Any player caught exploiting or abusing the bot (e.g., spamming commands/bypassing or exploiting) will be blacklisted.

Blacklist Information and Help

  • Blacklist Duration: Blacklists are typically temporary, especially for minor offenses like spamming commands or gaining a role without permission.
  • Permanent Blacklist: Major exploits, like gaining admin access, will result in a permanent blacklist.
  • Appealing a Blacklist: If you believe you’ve been unfairly blacklisted, DM @nmcli or an admin for assistance.
